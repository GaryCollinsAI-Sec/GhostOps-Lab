# GhostOps Lab

GhostOps Lab is a personal cybersecurity operations lab designed to practice **offensive security, reconnaissance, and penetration testing** using a Raspberry Pi target and a Kali Linux attacker VM for educational purposes.

---

## Objectives

The main objectives of GhostOps Lab are:

- Build a small, self-contained penetration testing lab environment.
- Practice reconnaissance, vulnerability scanning, and exploitation techniques.
- Simulate a real-world offensive security workflow in a safe, isolated environment.
- Document methodology and results for learning and portfolio purposes.

---

## Skills Learned

Through this lab, I gained hands-on experience with:

- Network reconnaissance and mapping
- SSH and HTTP-based attacks
- Vulnerable web application exploitation
- Traffic capture and analysis using Wireshark/tcpdump
- Red team workflow methodology
- Documenting attack methodology for portfolio presentation

---

## Tools Used

| Tool | Purpose |
|------|---------|
| Kali Linux | Attacker platform for pentesting |
| Nmap | Network discovery and service enumeration |
| Burp Suite | Web application proxy and vulnerability testing |
| OWASP Juice Shop | Vulnerable web application target |
| Raspberry Pi 3B+ | Hardware target platform |
| Wireshark / tcpdump | Network traffic capture and analysis |
| SSH | Secure remote access to the target |
